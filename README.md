This project is a YouTube-inspired Video Media Player built with React and Vite,
allowing users to browse, search, and watch videos dynamically via the YouTube v3
API using Axios and TanStack Query for efficient data fetching, caching, and error 
handling. It features a responsive, modular interface with a Navbar, Sidebar, Video
Feed, Video Details, and Channel Details pages, using reusable components like VideoCard,
ChannelCard, and VideoPlayer (fullscreen supported), while securing the API key through
environment variables and ensuring a seamless user experience across devices.
